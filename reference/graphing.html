Graphing¶

Graph¶

class Graph(func, x_range=(-5, 5), y_range=None, x=120, y=60, plot_width=1440, plot_height=840, **styling)¶

Bases: VCollection

Full plot with axes, ticks, labels, and a plotted function curve.

Parameters:

  • func (callable) – Function to plot (f(x) -> y).

  • x_range (tuple) – (x_min, x_max).

  • y_range (tuple) – (y_min, y_max) or None for auto.

  • x (float) – Plot area left edge.

  • y (float) – Plot area top edge.

  • plot_width (float) – Plot area width.

  • plot_height (float) – Plot area height.

Adding Curves

add_function(func, **styling)¶

Add another function curve to the plot. Returns the Lines object.

plot(func, num_points=200, **styling)¶

Plot a function. Returns a Lines object.

plot_line_graph(x_values, y_values, **styling)¶

Plot a line graph from data points.

Coordinate Conversion

coords_to_point(x, y)¶

Convert math coordinates to SVG coordinates.

Returns:

(svg_x, svg_y)

input_to_graph_point(x, func)¶

Get the SVG point on a function at x.

Returns:

(svg_x, svg_y)

graph_position(func, x_attr)¶

Return a lambda that gives the SVG position of a point following a function curve as x_attr changes over time. Useful with Coor.set_onward().

x_val = attributes.Real(0, 0)
x_val.move_to(0, 3, 5)
dot.c.set_onward(0, graph.graph_position(func, x_val))

Decorations

Shaded area under (or between) curves. Returns a Polygon.

get_vertical_line(x, y_val=None, **styling)¶

Vertical line at x. Returns a Line.

get_riemann_rectangles(func, x_range, dx=0.1, **styling)¶

Riemann sum rectangles under a curve. Returns a VCollection.

NumberLine¶

class NumberLine(x_range=(-5, 5, 1), length=720, x=240, y=540, tick_size=12, include_numbers=True, include_arrows=True, **styling)¶

Bases: VCollection

Horizontal number line with tick marks and optional labels.

Parameters:

  • x_range (tuple) – (start, end, step) or (start, end) with auto step.

  • length (float) – Line length in pixels.

  • tick_size (float) – Tick mark height in pixels.

  • include_numbers (bool) – Show tick labels.

  • include_arrows (bool) – Show endpoint arrows.
